How to RUN the file : 
 
 * Prerequisites
   You must have Python 3 and Git installed on your system.

Step 1: Clone the Repository
git clone https://github.com/Aymzsy/Algorithmic-Risk-Model.git
cd Algorithmic-Risk-Model

Step 2: Set up the Environment
python -m venv venv

Step 3: Install Dependencies
pip install pandas numpy yfinance matplotlib streamlit

Step 4: Launch the Application
python -m streamlit run Risk_Model.py
